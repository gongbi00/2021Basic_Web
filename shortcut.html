Intellij IDEA 단축키

  • Fix it

    • Alt + Enter

    - 현재 컨텍스트에 대한 액션을 표시하여 뭐든지 수정하는데 사용
    <오류/경고/제안이 표시된 경우>
    - 문제에 대한 수정 제안 목록 표시
    <오류/경고/제안이 없는 경우>
    - 인텐션 액션 및 활성화 되었지만 경고 설정은 안된 검사 표시

    * 실행화면

  • Next Error

    • F2

    - 다음 오류/경고/제안으로 점프

    * 실행화면

  • Project Window

    • MacOS : ⌘1
    Win or Linux : Alt + 1

    - 프로젝트 창이 열리고 프로젝트 창에 포커스 위치

    * 실행화면

  • Focus On Editor

    • ESC

    - 에디터에 포커스 위치

    * 실행화면

  • View Recent Files

    • MacOS : ⌘E
    Win or Linux : Ctrl + E

    - 최근 실행했던 파일 확인

    * 실행화면

  • Go To Declaration

    • MacOS : ⌘B
    Win or Linux : Ctrl + B

    - 심볼의 선언으로 이동

    * 실행화면

  • Find Usages

    • Alt + F7

    - 어떤 항목이 사용된 모든 곳 표시

    * 실행화면

  • Run Anything

    • Tap Ctrl Twice

    - 어떤 항목이든 실행
    - Run Anything 창이 열림
    - 기본적으로 Run Anything 창에는 최근 실행 구성 목록 표시

    * 실행화면

  • Extend / Shrink Selection

    • MacOS : ⌥↑ or ⌥↓
    Win or Linux : Ctrl + W or Shift + Ctrl + W

    - 커서 근처의 코드 선택 영역을 확대/축소

    * 실행화면

  • Add / Remove Comment

    • MacOS : ⌘/
    Win or Linux : Ctrl + /

    - 해당 코드 줄이 주석으로 처리
    - 주석으로 처리된 경우면 주석처리 해제

    * 실행화면

  • Complete Current Statement

    • MacOS : ⇧⌘↩︎
    Win or Linux : Shift + Ctrl + Enter

    - 현재 구문 완성

    * 실행화면

  • Reformat File

    • MacOS : ⌥⌘L
    Win or Linux : Ctrl + Alt + L

    - 해당 프로젝트의 표준에 맞게 현재 파일의 서식 지정
    - 파일에서 변경된 줄 또는 전체 파일의 서식을 지정하도록 설정 가능

    * 실행화면

  • Refactor

    • MacOS : ⌃T
    Win or Linux : Shift + Ctrl + Alt + T

    - 사용 가능한 리팩토링 옵션 표시 (심볼/선택영역)

    * 실행화면

  • Find Action

    • MacOS : ⇧⌘A
    Win or Linux : Shift + Ctrl + A

    - Intellij IDEA에서 모든 액션 검색
    - 설정도 검색 가능
    - 도구 창을 검색하고 열기 가능

    * 실행화면

  • Search Everywhere

    • Tap Shift Twice

    - 검색 상자를 통해 전체 검색

    * 실행화면
